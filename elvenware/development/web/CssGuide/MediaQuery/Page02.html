Page 3: Media Queries in the Head Element

This page must use media queries to ensure that different CSS files are loaded depending on the resolution of the device on which it is displayed. See the code for this page, and also this Elvenware page, which discussess a number of the details.

<link
  media="only screen and (min-width: 800px)"
  rel="stylesheet"
  type="text/css"
  href="Styles/Page04.css">

<link
  media="only screen and (max-width: 799px)"
  rel="stylesheet"
  type="text/css" 
  href="Styles/Page02.css">
