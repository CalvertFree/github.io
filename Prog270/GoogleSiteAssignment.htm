We will do three things:

  • Create a page on Google Site called Prog270Oct08

When you press the submmit button the assignment, you will turn in two things:

  • The URL of the page you created on Google Site
  • The markdown that you used to create your page.

Find a file URL from the WikiMedia Commons.

When you find the image you want. Get its URL. Go, for instance, to this page. Click on the Use this file on the web link. Grab the File URL. Embed it in your markdown, using syntax like this:
