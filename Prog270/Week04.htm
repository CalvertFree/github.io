Here is an example of how to combine three different "looks" for a page in a single CSS file. The appearance of the page changes as you resize the browser:

/* Default */
body {
    background-color: #0088FF;
    color: #FFFFFF;
}

/* Medium */
@media screen and (max-width: 1024px) {
    body {
        background-color: #00FF00;
        color: #006600;
    }
}

/* Small */
@media screen and (max-width: 520px) {
    body {
        background-color: #FF0000;
        color: #440000;
    }
}
